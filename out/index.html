options.js

The options page for the extension, runs on first run of the extension, when called from the settings link in the popup, and when the options link is clicked on the the chrome://extensions page The page allows for setting and updating the token and the idle time and saves these settings to local storage.

popup.js

The popup page for the extension, runs when the extensions toolbar button is clicked on The page shows the status for the idle time, the last idle time, and the running status It also provides a settings link that will open the options.html page
